# 🚀 Sistema Diplomats V1

Sistema de gestión desarrollado para **Diplomats Servicios Técnicos Integrales**.

## Objetivo

Crear un sistema simple, rápido y profesional para administrar:

- 📅 Agenda
- 👷 Técnicos
- 📋 Órdenes de trabajo
- 📦 Stock
- 🧰 Herramientas
- 📸 Antes y Después
- ✍️ Firma del cliente
- 👥 Clientes
- 💰 Presupuestos
- 🔔 Notificaciones

---

## Roles del sistema

### 👑 Administrador
Yamil

Funciones:
- Administrar toda la empresa.
- Aprobar trabajos.
- Ver estadísticas.
- Administrar usuarios.
- Controlar stock.

---

### 👩 Encargada
Ailén

Funciones:
- Revisar trabajos.
- Administrar stock.
- Preparar materiales.
- Aprobar revisiones.

---

### 👷 Instalador
Arian

Funciones:
- Ver agenda.
- Iniciar trabajos.
- Registrar Antes y Después.
- Cargar materiales utilizados.
- Solicitar firma del cliente.
- Finalizar trabajos.
